Bahasa Pemrograman R

Bahasa pemrograman R merupakan salah satu bahasa pemrograman yang cukup populer digunakan dalam data science bersama Python. R menyediakan berbagai teknik statistika (permodelan linier dan nonlinier, uji statistik klasik, analisis deret waktu, klasifikasi, klasterisasi, dan sebagainya) serta grafik. Kekuatan besar dari R yang lain adalah fasilitas grafiknya, yang menghasilkan grafik dengan kualitas publikasi yang dapat memuat simbol matematika. Bahasa R kini menjadi standar de facto di antara statistikawan untuk pengembangan perangkat lunak statistika, serta digunakan secara luas untuk pengembangan perangkat lunak statistika dan analisis data.
Di bawah ini adalah tahapan instalasinya.

Cara Instalasi Bahasa R di Ubuntu

Langkah 1 : Pertama, kamu harus melakukan update terhadap package list menggunakan command berikut:

$sudo apt update && sudo apt upgrade

Langkah 2 : Selanjutnya, mengunduh package yang dibutuhkan untuk menjalankan R di Ubuntu dengan menggunakan command:

$sudo apt install dirmngr gnupg apt-transport-https ca-certificates software-properties-common

Kebetulan pada device saya sudah terinstal semua package-nya.

Langkah 3 : Setelah memasang dependensi yang dibutuhkan, sekarang autentikasi package dengan:

$sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E298A3A825C0D65DFD57CBB651716619E084DAB9

Langkah 4 : Tambahkan repositori CRN dengan command berikut:

$sudo add-apt-repository 'deb https://cloud.r-project.org/bin/linux/ubuntu focal-cran40/'

Penambahan CRN repository sudah berhasil.

Langkah 5 : Setelah pemasangan di atas sudah selesai, sekarang install bahasa R dengan command:

$sudo apt install r-base

Langkah 6 : Setelah bahasa R terpasang, kamu bisa menjalankan R di terminal:

$sudo -i R

Menjalankan R di terminal (console) sudah berhasil.
